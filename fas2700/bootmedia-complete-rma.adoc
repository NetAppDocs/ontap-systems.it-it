= Restituire il componente guasto a NetApp - AFF A220 e FAS2700
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Restituire la parte guasta a NetApp, come descritto nelle istruzioni RMA fornite con il kit. Vedere la https://mysupport.netapp.com/site/info/rma["Restituzione e sostituzione delle parti"] pagina per ulteriori informazioni.

Il sistema FAS2700 supporta solo procedure di ripristino manuale tramite supporti di avvio. Il ripristino automatico del supporto di avvio non è supportato.
