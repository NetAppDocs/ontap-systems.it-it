= Restituire il componente guasto a NetApp - FAS9000
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Restituire il componente guasto a NetApp, come descritto nelle istruzioni RMA fornite con il kit. Vedi il https://mysupport.netapp.com/site/info/rma["Restituzione e sostituzione delle parti"] pagina per ulteriori informazioni. Il sistema FAS9000 supporta solo procedure di ripristino manuale tramite supporti di avvio. Il ripristino automatico del supporto di avvio non è supportato.
